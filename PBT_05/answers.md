# PBT 05 - Responsive & SCSS

A1: <meta name="viewport" content="width=device-width, initial-scale=1.0">. width=device-width lay do rong thiet bi, initial-scale=1 dat muc zoom ban dau. Thieu the nay mobile render nhu desktop thu nho. Mobile-first viet CSS mac dinh cho mobile va min-width de nang cap; desktop-first viet mac dinh desktop va max-width de giam. Mobile-first thuong tot hon vi uu tien thiet bi nho va CSS gon.

A2: 576px small phone ngang, 768px tablet, 992px laptop nho, 1200px desktop, 1400px desktop lon. Luoi san pham tuong ung 1,2,3,4,5 cot.

A3: 375px -> 100%; 600px -> 540px; 800px -> 720px; 1000px -> 960px; 1400px -> 1140px.

A4: SCSS co variables, nesting, mixins, extend. Browser không doc SCSS vi day la preprocessor syntax, can compile sang CSS bang sass. Lenh: sass scss/style.scss scss/style.css.
